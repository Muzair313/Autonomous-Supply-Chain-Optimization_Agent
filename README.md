# Autonomous-Supply-Chain-Optimization_Agent
Agentic LLM-Based Supply Chain Optimization System
Built an end-to-end supply chain decision system using demand forecasting (M5 dataset), simulation modeling, and LLM-based multi-agent negotiation (via Ollama). Designed decentralized agents (retailer, distributor, manufacturer) to collaboratively optimize ordering decisions, reducing the bullwhip effect and improving inventory stability. Deployed using FastAPI with an interactive dashboard for real-time simulation.
